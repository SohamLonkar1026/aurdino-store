# Aurdino Store

Aurdino Store is a specialized e-commerce platform tailored for campus communities to seamlessly order and receive Arduino microcontrollers and electronic components.

## Problem Solved
Procuring electronics components for projects often introduces delays that stall development. Aurdino Store solves the critical problem of component accessibility by offering **on-campus delivery within 24 hours**. By maintaining local inventory and managing logistics right at the institution, students and faculty can continue their hardware projects with minimal downtime.

## Features
- **Fast Delivery:** On-campus delivery guaranteed within 24 hours.
- **Full-stack Platform:** Built with React, Vite, and tailwindcss on the frontend.
- **Robust Backend:** Powered by Node.js, Express, and a Drizzle ORM integrated database.
- **Real-time updates:** Integrated WebSocket connections for instant updates and reliable state management.

## Tech Stack
- Frontend: React, Vite, TailwindCSS, Radix UI Themes
- Backend: Express, Node.js, WebSockets
- Database: SQLite/Neon Database, Drizzle ORM

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```
